Online Shop Web App

Welcome to my Shop. This application allows users to browse products, filter them by name, and manage a shopping cart dynamically.

## 🚀 Features

* **Product Display:** Dynamic list of items
* **Real-time Filtering:** Easily search for your favorite products
* **Shopping Cart Management:** Add or remove products and view the total number of items
* **Responsive Design:** Adapted to different screen sizes
* **Styling Architecture:** Modular SASS structure for scalable styles

## 🛠️ Technologies Used

* HTML5
* SASS (SCSS): Modular styling architecture
* JavaScript (ES6+): Functional logic for state management
* Vite: Build tool and development server

## 📂 Project Structure

The project follows a professional structure to ensure maintainability:

```
├── public/          # Static assets
├── src/
│   ├── js/          # Main logic (main.js)
│   └── scss/        # SASS styles
├── index.html       # Main structure
└── vite.config.js   # Vite configuration
```
